# TCHCustomDev JWKS Hosting

This repository hosts the JSON Web Key Set (JWKS) for Epic VendorServices integration.

**JWKS Endpoint:** https://ofomamj.github.io/tchcustomdev-jwks/.well-known/jwks.json

## Usage

Copy the JWKS URL above and paste it into the Epic VendorServices "Non-Production JWK Set URL" field.
